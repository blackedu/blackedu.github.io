# 玩点数学
这是我上大学时思考的一个小问题，简化一下是这样的：
> 如果你有100m的栅栏，去围一片草地，怎么围才能使得占有草地面积最大？

先来分析一下：
1. 没人愿意围成凹图形，根据根据对称性，凹的部分转为凸出面积肯定会更大；
2. 围成矩形的话，我们知道周长相等情况下，正方形面积最大可以达到225平方米；
3. 如果围成圆的话，100m就是周长，于是面积约为253.3平方米。
